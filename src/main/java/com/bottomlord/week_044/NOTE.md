# Interview_1709_第k个数
## 题目
有些数的素因子只有 3，5，7，请设计一个算法找出第 k 个数。注意，不是必须有这些素因子，而是必须不包含其他的素因子。例如，前几个数按顺序应该是 1，3，5，7，9，15，21。

示例 1:
```
输入: k = 5

输出: 9
```
## 解法
### 思路
动态规划：
- 初始化3个指针：
    - p3：该指针指向的数字只乘以3
    - p5；该指针指向的数字只乘以5
    - p7：该指针指向的数字值乘以7
- `dp[i]`：第i个丑数
- 状态转移方程：
    - `dp[i] = min(dp[p3] * 3, dp[p5] * 5, dp[p6] * 6)`
    - 谁是最小值，谁的指针+1
- 初始化：
    - dp[0] = 1
    - p3 = 0
    - p5 = 0
    - p7 = 0
- 结果：返回`dp[k - 1]`
